# Minimal Linktree-style portfolio

Built with **Next.js (App Router)** + **Tailwind CSS** and includes a **GitHub contributions calendar** for `@atfbcs`.

## Local dev

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Edit content

- **Main page**: `app/page.tsx`
- **GitHub calendar**: `components/GithubCalendar.tsx`
